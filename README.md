# AI Analytics Portfolio

Привет! Это моё портфолио проектов по анализу данных и машинному обучению.

## 📊 Проекты

1.  **A/B Тестирование**  
    *   Оценка эффективности маркетинговых кампаний с помощью t-test и chi-square теста.  
    *   Результаты оформлены в виде отчётов и графиков.

2.  **Анализ оттока клиентов**  
    *   Построена модель прогнозирования (CatBoost) с ROC-AUC 0.85.
    *   Выявлены ключевые факторы ухода клиентов.
    *   Результаты интегрированы в Telegram-бота.

3.  **Power BI Дашборд**  
    *   Модель "Звезда", DAX-меры (YTD, MoM, динамика).
    *   Настроена безопасность на уровне строк (RLS).

4.  **SQL Аналитика (PostgreSQL)**  
    *   Сложные запросы: оконные функции, CTE.
    *   Анализ 80 000 записей клиентов банка.
