## 📁 Проекты

### 1. A/B тестирование (маркетинг)
- **Задача:** Оценить эффективность рекламной кампании  
- **Что сделано:** t-test, chi‑square test, доверительные интервалы  
- **Результат:** Выявлен статистически значимый рост конверсии (p‑value = 0.03)  
- **Технологии:** Python, SciPy, Pandas, Matplotlib  
- 📂 [Код →](./ab_tests)

### 2. Power BI дашборд (анализ продаж)
- **Задача:** Визуализировать продажи по категориям, менеджерам, времени  
- **Что сделано:** Модель «Звезда», меры DAX (YTD, MoM), RLS (безопасность)  
- **Результат:** Интерактивный дашборд для топ‑менеджмента  
- **Технологии:** Power BI, DAX, PostgreSQL  
- 📂 [Дашборд →](./power_bi)

### 3. Big Data (PySpark)
- **Задача:** Обработать 10 млн строк (сравнение Pandas vs Spark)  
- **Что сделано:** Оптимизация партиций, Spark SQL, анализ производительности  
- **Результат:** Spark справился там, где Pandas упал по памяти  
- **Технологии:** PySpark, Spark SQL  
- 📂 [Код →](./spark)

### 4. Docker + Telegram‑бот
- **Задача:** Упаковать Telegram‑бота в контейнер  
- **Что сделано:** Dockerfile, сборка образа, запуск контейнера  
- **Результат:** Бот работает изолированно, легко деплоить  
- **Технологии:** Docker, Python, Telegram API  
- 📂 [Код →](./docker)

### 5. RAG‑агент (поиск по документам)
- **Задача:** Ответы на вопросы по загруженным PDF  
- **Что сделано:** Векторная БД (ChromaDB), эмбеддинги (multilingual), интеграция с Groq  
- **Результат:** Чат с документами в консоли  
- **Технологии:** ChromaDB, Sentence Transformers, Groq API  
- 📂 [Код →](./rag_bot)
